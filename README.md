# TabStash 🗂️

TabStash 是一個輕量、美觀且強大的瀏覽器分頁管理擴充功能（Chrome / Edge 適用）。幫助你一鍵收集凌亂的分頁，保持瀏覽器整潔，並提供方便的管理介面。

## ✨ 核心功能

- **一鍵收集**：點擊按鈕即可將目前所有分頁收納起來（支援預設排除鎖定中的分頁）。
- **智慧分組**：收集的 Sessions 會自動按照時間（今天、昨天、本週、更早）進行分組，一目瞭然。
- **美觀介面**：採用現代化的暗色系玻璃擬態（Glassmorphism）設計與流暢動畫。
- **全頁面管理**：支援搜尋、篩選與完整管理功能（包含重新命名、單一分頁移除、還原）。
- **資料安全**：支援手動匯出/匯入 JSON 格式的備份檔案。
- **雲端同步**：（開發中）未來將支援 Google Drive 與 OneDrive 跨裝置同步。

## 🚀 如何安裝 (手動載入)

由於目前專案處於開發階段，尚未發布至擴充功能商店，請依照以下步驟「手動安裝」至您的瀏覽器：

### Chrome 瀏覽器

1. **下載程式碼**：將本專案 Clone 到本機或下載 ZIP 檔並解壓縮。
   ```bash
   git clone https://github.com/larryyangsen/tab-manger.git
   ```
2. **開啟擴充功能管理頁面**：
   在網址列輸入 `chrome://extensions/` 並前往。
3. **開啟開發人員模式**：
   在頁面右上角，開啟 **「開發人員模式 (Developer mode)」** 的開關。
4. **載入擴充功能**：
   點擊左上角的 **「載入未封裝項目 (Load unpacked)」** 按鈕。
5. **選擇資料夾**：
   選擇你剛剛下載的 `tab-manager` (或 `tab-manger`) 資料夾。
6. **完成！** 🎉
   建議將 TabStash 從拼圖圖示 🧩 釘選到工具列上，方便隨時使用。

### Edge 瀏覽器

1. **下載程式碼**：將本專案 Clone 到本機或下載 ZIP 檔並解壓縮。
2. **開啟擴充功能管理頁面**：
   在網址列輸入 `edge://extensions/` 並前往。
3. **開啟開發人員模式**：
   在頁面左下角，開啟 **「開發人員模式」** 的開關。
4. **載入擴充功能**：
   點擊右上角的 **「載入解壓縮的擴充功能」** 按鈕。
5. **選擇資料夾**：
   選擇你剛剛下載的資料夾。
6. **完成！** 🎉

## 🛠️ 技術架構

- **純原生開發 (Vanilla JS)**：無需打包工具 (No Build Step)，直接使用 ES Modules。
- **Manifest V3**：符合最新的 Chrome 擴充功能安全與效能標準。
- **Service Worker**：背景處理分頁收集與資料管理。
